Getting Started With Gmail

The challenge

Design and write documentation for the Gmail email client, written for an audience that was familiar with email clients, but not with Gmail in particular.

My resolution

The documentation that I wrote was split into four sections: one that explained the benefits of Gmail, two that covered the main setup process, and a third that dealt with setting up Gmail on your mobile device. This way, the user had all of the information they needed to both ensure that Gmail was right for them and then complete at least the initial setup so that they could get started using Gmail right away.

New Interview FAQ

My resolution

The article that I wrote was split into a couple sections: first things to do when you think it's time to find a new job; things you should do before you apply and questions to ask your interviewer, as well as a small section on how to handle interviews during work hours.

Installing an IRC Client On Ubuntu

The challenge

Rebuild the installation instructions for HexChat IRC to be more inclusive toward non-computer-inclined end users. Write documentation that includes all three methods of installation in a way that makes them easy to understand even if users are not familiar with things like source code compilation.

My resolution

I described each of the three methods of installation and broke them down in ways that were beginner-friendly, including walking through how to use the command line and compile using "tar".
